## 句型

```c
// 1.1
// 입니다 用于表示事物的身份、状态或指定某个事物。通常用于正式场合。

저는 학생입니다.
我是学生。
```

```c
// 온/는 用于体词后面的助词，表示句子所陈述的主体或要讨论的话题，体词以辅音结尾用온，原因结尾用는。

서울은 한국의 수도입니다.
首尔是韩国的首都。

저는 한국 사람입니다.
我是韩国人。
```

```c
// 온/는 有时表示对比或强调

동생은 키가 큽니다. 저는 키가 작습니다.
弟弟个子高。我个子矮。
```


```c
// 1.2
// 입니까? 表示疑问，"입니다"的疑问句形式，"?"置于末尾。

이분이 어느 나라 사람입니까?
这位是哪个国家的人？
```

```c
// 1.3
// 이/가 아나다 和体词连用，是"이다"的否定形式。如果体词以辅音结尾用"이 아니다"，以元音结尾用"가 아니다"

이것은 교과서가 아닙니다.
这不是教科书。

저는 일본 사람이 아닙니다
我不是日本人。
```

```c
// 1.4
// -습니다/ㅂ니다 和谓词词干连用，向对方描述一个具体的动作或状态。如果谓词词干以元音结尾用"-ㅂ니다"，以辅音结尾用"-습니다"。

학교에 갑니다.
去学校。

저는 키가 큽니다.
我个子很高。

의자가 없습니다.
没有椅子。
```

```c
// -습니까/ㅂ니까? 和谓词词干连用，表示对具体的动作或状态的疑问。如果谓词词干以元音结尾用"-ㅂ니까"，以辅音结尾则用"-습니까"。

무엇을 합니까?
做什么？

어디에 갑니까?
去哪里？

한국 친구가 없습니까?
没有韩国朋友吗？
```

```c
// 읽습니다.阅读。

읽습니까?
阅读吗？

// 씁니다.写。

씁니까?
写吗？
```

```c
// 2.2
// 도 “也” 用于体词后面的助词，表示列举相同或相似的事物。与主格助词“이/가”、宾格助词“을/를”连用时，主格助词、宾格助词要省略。

책상이 있습니다 의자도 있습니다. 시계도 있습니다.
有书桌，也有椅子，还有钟表。

책이 많습니다. 그림도 많습니다.
书很多，画也很多。
```

```c
// 当它与“에、 에서”连用表示场所或位置时，“에、 에서”不能省略。

산에 갑니다. 바다에도 갑니다
我去爬山，也去海边。

도서관에서 공부합니다. 집에서도공부합니다.
我在图书馆学习，也在家学习。
```

```c
// 있다, 없다 “있다”表示“有，在”，“없다”表示“没有，不在”

친구가 있습니다.
我有朋友。

그림이 없습니다.
我有画

질문이 있습니까?
你有问题吗？

돈이 없습니까?
你没有钱吗？
```

```c
// 如果主语是需要表示尊敬的人，用“계시다”，而不用“있다”。

선생님이 계십니다.
老师在。

아버지가 안 계십니다.
爸爸不在。
```

```c
// 2.3
// 에 和体词连用的助词，在这里表示主体的存在住所，通常和“있다”、“없다”连用。

친구가 미국에 있습니다.
朋友在美国。

고양이가 방에 없습니다.
猫不在房间里。

지갑에 돈이 많습니다.
钱包里有很多钱。

어디입니까 和 어디에 있습니까 都可以
```

```c
// 2.4
// 하고 “和” 和体词连接的助词，连接两个或两个以上的体词。意思和“그리고”相近。

책하고 공책이 있습니다.
有书和笔记本。

한국말하고 일본말을공부합니다.
我学习韩语和日语。
```

```c
// 3.1
// 을/를 附于体词后面的宾格助词，表示句子的宾语。如果体词以辅音结尾用“을”，以元音结尾用“를”。

이 아이는 빵을 좋아합니다.
这孩子喜欢吃面包。

나는 친구를 기다립니다.
我在等朋友。
```

```c
// 3.2
// -으시/시- 尊称词尾，一般用于谓词词干之后，表示对主语的尊敬。如果词干以辅音结尾用“-으시-”，以元音结尾用“-시-”。表示敬语的后缀“-으시/시-”通常和敬语主格助词“께서”一起使用。

어머니 께서 가십니다.
妈妈去。

아버지께서는 신문을 읽으십니다.
爸爸读报纸。
```

```c
// 有些词变成敬语形式不用加“-으시”，而换成自身带有尊敬意义的词。

먹다 → 잡수시다/드시다
吃

있다 → 계시다
```

```c
// 5.2
// "-이에요/예요"是"입니다/입니까?"的非格式体表现形式，其基本形态也是"-이다"。接在体词(名词、代词、数词)之后，表示" 是..."，也是敬语。不同之处在于，"-이에요/예요"比"입니다/입니까?"的尊敬和正式程度浅一点，一般用于口语。

形态变化：

♦前面体词为开音节时，接"예요"

例 바나나+예요→바나나예요.

♦前面体词为闭音节时，接"이에요"

例 수박+이에요→수박이에요.
```
